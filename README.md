# Projeto Lakehouse com PySpark

Projeto acadêmico desenvolvido com PySpark utilizando duas tecnologias modernas de Data Lakehouse:

- Delta Lake
- Apache Iceberg

## Dataset utilizado

Campeonato Brasileiro (CSV)

## Objetivos

- Construção de pipeline Bronze / Silver / Gold
- Processamento distribuído com Spark
- Armazenamento transacional com Delta Lake
- Tabelas analíticas com Iceberg

## Tecnologias

- Python
- PySpark
- Jupyter Notebook
- Delta Lake
- Apache Iceberg

## Estrutura

- `notebooks/delta_lake.ipynb`
- `notebooks/iceberg.ipynb`
