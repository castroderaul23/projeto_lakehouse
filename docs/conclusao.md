# Conclusão

O projeto permitiu aplicar conceitos modernos de Engenharia de Dados utilizando arquitetura Lakehouse.

Durante o desenvolvimento foram utilizados:

- Apache Spark
- PySpark
- Delta Lake
- Apache Iceberg
- arquitetura Bronze/Silver/Gold

O projeto demonstrou na prática como estruturar pipelines analíticos modernos utilizando tecnologias amplamente utilizadas no mercado.
