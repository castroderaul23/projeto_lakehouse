# Dataset Utilizado

O projeto utiliza um dataset contendo informações históricas do Campeonato Brasileiro de 2005 a 2024.

## Principais colunas

- temporada
- data
- estádio
- árbitro
- público
- time mandante
- time visitante
- gols
- estatísticas da partida

## Objetivo do dataset

O dataset foi utilizado para demonstrar:

- ingestão de dados
- limpeza
- transformações
- persistência em arquitetura Lakehouse
